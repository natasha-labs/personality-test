# Тест «Какая стратегия управляет вами?»

Рабочая статическая версия без сборщика и без внешних изображений.

## Структура

- `index.html` - все экраны теста
- `assets/css/styles.css` - дизайн и CSS-иллюстрации
- `assets/js/data.js` - 20 вопросов и 5 стратегий
- `assets/js/app.js` - переходы, подсчет и результат

## Публикация

Замените содержимое репозитория этими файлами целиком.
Для Cloudflare Pages:

- Build command: пусто
- Build output directory: `.`

Последовательность экранов:

Главный экран → Перед началом → Вопросы → Анализ → Результат → Подробности → Следующий шаг.
