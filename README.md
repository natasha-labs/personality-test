# Personality Test V2

Статический лендинг с психологическим тестом из 20 вопросов и 5 стратегий.

## Файлы контента

- `assets/js/questions.js` - все вопросы теста. Их можно редактировать отдельно от логики.
- `assets/js/data.js` - описания стратегий и варианты ответов.
- `assets/js/app.js` - логика теста, подсчет результата и переходы между экранами.
- `assets/css/styles.css` - весь дизайн.

## Публикация на Cloudflare Pages

1. Замените в GitHub старые файлы полными файлами из этой версии.
2. Cloudflare Pages автоматически запустит новый deployment после commit в ветку `main`.
3. Build command оставьте пустым.
4. Build output directory: `.`

## Важно

Тест не является диагностическим инструментом. Он показывает наиболее выраженную стратегию среди пяти исследуемых вариантов.
