# Personality Test V1

Статический лендинг с психологическим тестом из 20 вопросов.

## Cloudflare Pages
1. Создайте репозиторий GitHub.
2. Загрузите содержимое этой папки.
3. В Cloudflare Pages выберите Connect to Git.
4. Framework preset: None.
5. Build command: пусто.
6. Build output directory: `/`.
7. Deploy.

Вопросы и тексты результатов находятся в `assets/js/data.js`.
